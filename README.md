# pedestro-detecto
Attempt at training a CNN for binary classification on the CIFAR100 dataset to detect people.

## Current Results:
```
Accuracy of the network on the 10000 test images: 91 %
Accuracy of not_person : 92 %
Accuracy of person : 62 %
```
